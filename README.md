## GitHub Finder
  Aplicativo para fazer procura de perfis do GitHub.

## Utilizações
  <p>A linguagem utilizada foi React + Typescript</p>
  <p>Também foi utilizado as seguintes bibliotecas:</p>
  <ul>
    <li>react-icons</li>
    <li>react-hot-toast</li>
  </ul>
